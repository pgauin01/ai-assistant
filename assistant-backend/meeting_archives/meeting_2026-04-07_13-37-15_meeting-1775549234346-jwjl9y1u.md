### assistant
A closure is the combination of a function and the lexical environment that created it.  
When a function is defined, it captures references to variables from its surrounding scope. Those references stay alive even after the outer function has returned, allowing the inner function to access and modify those variables.

```js
function makeCounter() {
  let count = 0;          // variable in outer scope
  return function() {     // inner function forms a closure
    count += 1;           // accesses outer variable
    return count;
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2
```

Key points:

- **Lexical scoping**: the function remembers the environment where it was created.
- **Persistence**: the captured variables persist as long as the closure exists.
- **Encapsulation**: closures provide private state for modules or objects without using classes.
