what is closure in js

### assistant
A closure is a function that remembers the lexical scope in which it was created, even after that outer function has finished executing.  
It gives the inner function access to variables from its parent scope.

```js
function makeCounter() {
  let count = 0;          // count lives in makeCounter's scope
  return function() {     // inner function forms a closure
    return ++count;       // can read/modify `count`
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2
```

Key points  
- The returned function retains a reference to `count`.  
- The outer function’s stack frame is not destroyed; the closure keeps the needed variables alive.  
- Closures enable data encapsulation, factories, and functional patterns.
