Cursor predicts the next snippet by running a Fill‑In‑The‑Middle model over the code you’ve opened.  
It first builds a deep context: parses the AST, grabs surrounding tokens, and reads all open files via the LSP.  
That context is fed to the model, which scores every possible continuation and picks the highest‑probability token sequence.

### 1. The Elevator Pitch  
Cursor’s autocomplete works by feeding a Fill‑In‑The‑Middle (FIM) prompt that combines precise AST context, open‑file tokens, and a tiny project‑wide retrieval slice into a LLM, then validates the suggestion with a fast “shadow compilation” step.

### 2. Core Mechanics  
The editor sends the current file’s AST and surrounding tokens via LSP. A lightweight BM25/vector index of the opened project is queried for the top‑k most relevant snippets, avoiding raw‑text overload that would hit token limits and cause phrase‑sensitivity hallucinations. The AST, local tokens, and retrieved snippets are concatenated into a FIM prompt; the model runs at low temperature with top‑p filtering, scoring continuations. Before displaying, a sandboxed parser/type‑checker runs on the output—any syntax error or type mismatch discards the candidate, dramatically cutting hallucinations.

### 3. Top Options & Trade-offs  
- **Pure token window** – simplest, but token limit kills recall and increases hallucination.  
- **AST‑only FIM** – improves precision, but misses cross‑file patterns.  
- **AST + retrieval (current)** – balances recall and latency; index sync adds modest CPU overhead.  
We start with AST‑only for tiny scripts; once the project exceeds ~200 files we enable retrieval to keep recall high without blowing latency.

### 4. Production Example  
A developer adds a new Django view. Cursor parses the function’s AST node, pulls the three most similar view implementations from the index, builds a FIM prompt, and the LLM returns a boilerplate `class‑based` view. The shadow compiler type‑checks the snippet; it passes, so the IDE inserts it instantly, delivering a context‑aware, low‑hallucination suggestion.
