✅ Build a JavaScript function that debounces another function (i.e., delays execution until after a specified time has passed since the last call).

### assistant
### Implementation Strategy
To implement a debounce function in JavaScript, we can use a combination of closures and the `setTimeout` function. The idea is to create a wrapper function that will hold the state of whether it has been called recently. If the wrapper function is called again before the timeout expires, it will reset the timer and start counting from zero again.

### Code
```javascript
// 1. Your feature implementation here

function debounce(func, wait) {
    let timeout;

    return function(...args) {
        clearTimeout(timeout);
        timeout = setTimeout(() => func.apply(this, args), wait);
    };
}

// 2. Example Usage:
const logMessage = () => console.log('Function called after a delay');

const debouncedLogMessage = debounce(logMessage, 500);

debouncedLogMessage(); // This call will not trigger the function immediately
setTimeout(() => {
    debouncedLogMessage(); // This call will trigger the function after 500ms
}, 300); // The first call is delayed by 300ms, but it won't trigger until 800ms (300 + 500)
```

In this implementation:
- We define a `debounce` function that takes two arguments: the function to debounce (`func`) and the delay time in milliseconds (`wait`).
- Inside `debounce`, we use a variable `timeout` to store the timeout ID returned by `setTimeout`.
- The returned function, when called, clears any existing timeout using `clearTimeout(timeout)`.
- It then sets a new timeout using `setTimeout(func.apply(this, args), wait)`, which calls the original function after the specified delay.
- This ensures that if the function is called multiple times within the debounce period, only the last call will trigger the actual execution of the function.

### assistant
### Implementation Strategy
The solution involves recursively traversing through each element of the input array. If an element is an array itself, it is further processed to flatten its contents. This process continues until all elements are flattened into a single array.

### Code
```javascript
function flattenArray(nestedArray) {
  let result = [];

  function traverse(arr) {
    for (let i = 0; i < arr.length; i++) {
      if (Array.isArray(arr[i])) {
        // If the current element is an array, recursively process it
        traverse(arr[i]);
      } else {
        // Otherwise, add the element to the result array
        result.push(arr[i]);
      }
    }
  }

  traverse(nestedArray);
  return result;
}

// Example Usage:
const sampleData = [1, [2, [3, 4]], 5];
console.log(flattenArray(sampleData)); // Output: [1, 2, 3, 4, 5]
```

This function `flattenArray` takes a nested array as input and returns a flattened version of it. The helper function `traverse` is used to recursively process each element in the array. If an element is an array, it calls itself with that element; otherwise, it adds the element to the result array.
