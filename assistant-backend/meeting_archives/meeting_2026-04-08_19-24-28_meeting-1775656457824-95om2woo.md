I built an Advanced RAG Pipeline that combines hybrid search with reranking to solve the critical problem of LLM hallucinations when querying complex academic documents. The system uses Python, LangChain, Google Gemini 1.5 Flash, FAISS for dense retrieval, BM25 for sparse retrieval, and HuggingFace's BGE-Reranker for cross-encoder reranking. The main technical challenge was that standard vector search would either miss exact keywords or feed the LLM too much noisy context, causing it to hallucinate answers. I solved this by implementing a two-stage retrieval architecture where FAISS and BM25 first pull the top 10 candidates in a 70-30 weighted ensemble, then a computationally heavy Cross-Encoder reranks them and only passes the absolute top 3 highest-fidelity chunks to the LLM. I also had to engineer advanced document parsing using UnstructuredIO to handle multi-column PDFs and complex academic layouts that standard extractors would scramble. The result was near-zero hallucination rates with sub-second response times, making it viable for mission-critical enterprise deployments where factual accuracy is absolutely essential.
