### user
what is closure in js

### assistant
A closure is the combination of a function and the lexical environment that created it.  
When a function is defined, it captures references to variables in its surrounding scope. Those references stay alive even after the outer function has returned, allowing the inner function to access them later.

```js
function makeAdder(x) {
  return function(y) {          // inner function
    return x + y;               // uses x from outer scope
  };
}

const add5 = makeAdder(5);      // x = 5 is captured
console.log(add5(10));          // 15
```

- The inner function keeps a reference to `x`.  
- `x` is not part of the global scope, yet it remains accessible.  
- This is useful for data privacy, currying, and factory functions.
